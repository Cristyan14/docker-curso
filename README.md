Docker Apache HTML

Projeto que utiliza Docker Compose para rodar uma aplicação web simples em Apache.

Estrutura

docker-apache-html/
├── app/
│   ├── index.html
│   ├── style.css
│   └── script.js
└── docker-compose.yml

Como executar

docker-compose up -d

Acesse: http://localhost:8080

Pré-requisitos

Docker

Docker Compose
